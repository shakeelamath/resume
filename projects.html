Project 1: Music Recommendation System based on Facial Features

Led a team of five in developing a music recommendation system that utilized facial feature recognition to suggest personalized music playlists.
Designed the user interface using Figma, ensuring a visually appealing and user-friendly experience.
Implemented the front-end of the mobile application using Flutter and Dart, creating an intuitive user interface for capturing facial features.
Utilized Python and machine learning techniques to train a facial recognition model on a dataset of music preferences.
Integrated the trained model with the Flutter application to generate music recommendations based on facial expressions and features.
Collaborated with team members to ensure seamless communication between the front-end and back-end components of the system.
Managed project tasks, deadlines, and version control using Git and GitHub.

Project 2: Client-side Property Search Web Application

Created a client-side web application that enables users to search properties based on various criteria such as type, price, bedrooms, date added, and postcode area.
Developed the search form using HTML5 and incorporated jQuery UI widgets for consistency, reliability, and accessibility.
Designed the user interface using Figma, ensuring a visually appealing and user-friendly experience.
Utilized JSON data to populate the property database and ensure the search functionality works without a server.
Expanded the initial JSON dataset by adding an additional five properties to cover a wider range of search results.

Project 3: Skin Consultation Centre Management System

Developed a program using Java to implement a system for managing a Skin Consultation Centre.
Implemented a console-based system for the manager to add and delete doctors, schedule and cancel consultations, and print and save them.
Designed a Graphical User Interface (GUI) to provide a user-friendly view of the doctor list, booking and editing consultations, and other relevant functionalities.
Implemented the GUI without using drag and drop tools, such as the Designer in NetBeans, but utilized external APIs for graphs and professional components if necessary.

Project 4: Redux Shopping app

I designed and developed a dynamic shopping application using cutting-edge technologies, specifically combining React, Redux, and Tailwind CSS.
This project seamlessly manages the state of the application using Redux, ensuring efficient data flow and a smooth user experience. The user interface is crafted using Tailwind CSS,
resulting in a responsive and visually appealing design. Through this application, users can effortlessly browse a variety of products, add them to their cart,
and experience real-time updates thanks to the Redux architecture. This project showcases my proficiency in modern web development, user interface design, and state management.
